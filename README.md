# Тех задание eKapusta

### Изначальное ТЗ:
Есть пять сущностей:
1. Post Office(почтовое отделение)
2. Envelope(конверт)
3. Letter(письмо)
4. Stamps(почтовые марки, мн.число)
5. Address(адрес) 
#### Опишите взаимосвязь между этими пятью сущностями,что бы отправить письмо, то есть вызвать метод send
